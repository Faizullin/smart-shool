Задачи:

Подключить LCD-дисплей к Arduino.

Написать код программы для вывода значения температуры на LCD-дисплей.

Проверить работу устройства, убедившись, что оно точно измеряет температуру и выводит ее на дисплей.

Добавить функциональность, например, настройку пределов температуры, чтобы устройство могло предупреждать пользователя о превышении или понижении заданных значений.

Практические навыки: поможет студентам практически ознакомиться с программированием на Arduino, подключением датчиков и выводом данных на LCD-дисплей. Оно позволит применить полученные знания в создании полезного устройства, а также развить навыки программирования и работы с аппаратными компонентами.

Подключение схемы:

Подключите датчик температуры DS18B20 к Arduino:

-  Пин VCC датчика подключите к 5V пину Arduino.

-  Пин GND датчика подключите к GND пину Arduino.

-  Пин данных (DQ) датчика подключите к выбранному цифровому пину Arduino (например, пин 2).

Подключите LCD-дисплей к Arduino:

-  Пин VCC дисплея подключите к 5V пину Arduino.

-  Пин GND дисплея подключите к GND пину Arduino.

-  Пины SDA и SCL дисплея подключите к соответствующим пинам Arduino (обычно A4 и A5 соответственно).

Обратите внимание, что схема подключения может отличаться в зависимости от используемых моделей датчика температуры и LCD-дисплея. Важно проверить и соответствовать документации и примерам кода, которые вы используете.

 

Программный код:

#include <OneWire.h>

#include <LiquidCrystal_I2C.h>

// Подключение датчика температуры по шине OneWire к пину 2

OneWire ds(2);

// Создание объекта для работы с LCD-дисплеем по I2C

LiquidCrystal_I2C lcd(0x27, 16, 2);

void setup() {

 // Инициализация LCD-дисплея

lcd.begin(16, 2);

lcd.setBacklight(HIGH);

// Проверка и инициализация датчика температуры

if (!ds.search()) {

 ds.reset_search();

  }

}

void loop() {

  byte i;

  byte present = 0;

  byte type_s;

  byte data[12];

  byte addr[8];

  float celsius, fahrenheit;

  if (!ds.search()) {

    ds.reset_search();

    delay(250);

    return;

  }

  if (OneWire::crc8(addr, 7) != addr[7]) {

    return;

  }

 // Проверка типа датчика температуры

  if (addr[0] == 0x10) {

    type_s = 1;

  } else if (addr[0] == 0x28) {

    type_s = 0;

  } else {

    return;

  }

  ds.reset();

  ds.select(addr);

  ds.write(0x44, 1);

  delay(800);

  present = ds.reset();

  ds.select(addr);

  ds.write(0xBE);

  for (i = 0; i < 9; i++) {

    data[i] = ds.read();

  }

  int16_t raw = (data[1] << 8) | data[0];

  if (type_s) {

    raw = raw << 3;

    if (data[7] == 0x10) {

      raw = (raw & 0xFFF0) + 12 - data[6];

    }

  } else {

    byte cfg = (data[4] & 0x60);

    if (cfg == 0x00) {

      raw = raw << 3;

    } else if (cfg == 0x20) {

      raw = raw << 2;

    } else if (cfg == 0x40) {

      raw = raw << 1;

    }

  }

  celsius = (float)raw / 16.0;

  fahrenheit = celsius * 1.8 + 32.0;

 // Очистка дисплея

  lcd.clear();

 // Вывод значения температуры на первую строку дисплея

  lcd.setCursor(0, 0);

  lcd.print("Temp: ");

  lcd.print(celsius);

  lcd.print(" C");

// Вывод значения температуры в Фаренгейтах на вторую строку дисплея

  lcd.setCursor(0, 1);

  lcd.print("Temp: ");

  lcd.print(fahrenheit);

  lcd.print(" F");

  delay

}

«Мониторинг температуры и влажности с использованием датчика DHT11 на Arduino»

Уровень:**

 

Цель: работа состоит в создании программы и аппаратного устройства, которые позволят мониторить и выводить текущие значения температуры и влажности с помощью датчика DHT11 на платформе Arduino.

Задачи:

Реализовать обновление данных: настроить периодическое обновление считанных значений для получения актуальных данных о температуре и влажности.

Проверить работу системы: загрузить программу на Arduino и убедиться, что данные о температуре и влажности корректно считываются и отображаются.

Добавить дополнительные функции (по желанию): реализовать возможность отслеживания изменений в температуре и влажности, вывод предупреждений при превышении определенных значений, запись данных в файл и другие дополнительные функциональности.

Практические навыки: поможет студентам использовать функции и методы для считывания данных о температуре и влажности с датчика DHT11 и их сохранения в переменных для дальнейшей обработки.

Подключение схем:

-  Пин VCC датчика (обычно красный провод) подключается к пину 5V на Arduino для обеспечения питания датчика.

-  Пин GND датчика (обычно черный провод) подключается к пину GND на Arduino для общей заземления.

-  Пин DATA датчика (обычно желтый или синий провод) подключается к выбранному цифровому пину на Arduino, например, пину 2.

Обратите внимание, что номера пинов могут отличаться в зависимости от вашей конкретной схемы и настроек подключения.

 

 

Программный код:

#include <DHT.h>

#define DHTPIN 2         // Пин DATA подключен к пину 2 на Arduino

#define DHTTYPE DHT11    // Используем тип датчика DHT11

DHT dht(DHTPIN, DHTTYPE); // Создаем объект dht для работы с датчиком

void setup() {

  Serial.begin(9600);     // Инициализируем последовательный порт для вывода данных

  dht.begin();            // Инициализируем датчик DHT11

}

void loop() {

  // Считываем значения температуры и влажности

float temperature = dht.readTemperature(); // Читаем температуру в градусах Цельсия

float humidity = dht.readHumidity();       // Читаем влажность в процентах

 // Выводим значения на последовательный порт

  Serial.print("Temperature: ");

  Serial.print(temperature);

  Serial.print(" °C");

  Serial.print("   Humidity: ");

  Serial.print(humidity);

  Serial.println(" %");

  delay(2000); // Задержка 2 секунды между чтениями

}

Обратите внимание, что перед использованием этого кода необходимо установить библиотеку DHT, которая предоставляет функции для работы с датчиком DHT11. Вы можете установить ее в Arduino IDE через меню "Скетч" -> "Подключить библиотеку" -> "Управление библиотеками" и найти "DHT sensor library" от Adafruit.

В данном коде датчик DHT11 подключается к пину 2 на Arduino (вы можете изменить это значение в соответствии с вашими подключениями). Затем в функции loop() происходит чтение температуры и влажности с помощью методов readTemperature() и readHumidity() соответственно. Полученные значения выводятся на последовательный порт с помощью функций Serial.print() и Serial.println().

Вы можете загрузить этот код на Arduino и открыть последовательный монитор (Serial Monitor) в Arduino IDE, чтобы видеть текущие значения температуры и влажности, которые будут обновляться каждые 2 секунды.

return; }

server.begin();

Serial.print("Server Started.\nLocal IP: ");

Serial.println(Ethernet.localIP()); }

 {

 EthernetClient client = server.available();

 if (client) { boolean currentLineIsBlank = true;

                 boolean sentHeader = false;

while (client.connected())

{ if (client.available())

{ char c = client.read();

      receiving = false;

      receiving = true;

      if (c == 'L')

      {

        Serial.print("Переключение вывода ");

        pin = client.parseInt();

        Serial.println(pin);

        digitalWrite(pin, !digitalRead(pin));

      }

      currentLineIsBlank = false;

    }

}

Кроме того, некоторые маршрутизаторы могут не поддерживать все необходимые функции для настройки переадресации портов и динамического обновления DNS. Если вы не имеете опыта в сетевом администрировании, рекомендуется пока оставаться при локальном доступе к wev-интерфейсу вашего Arduino.

Ethernet Shield обычно имеет разъем RJ45 для подключения сетевого кабеля и слот для карты SD для хранения данных или веб-страниц.
